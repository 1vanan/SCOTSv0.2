template<class state_type, class input_type>
class scots::AbstractionGB< state_type, input_type >

Computation of the transition function of symbolic model based on a growth bound.

See

  • the manual in manual.pdf
  • http://arxiv.org/abs/1503.03715 for theoretical background

computes the transition function

Parameters
transition_function- the result of the computation
system_post- lambda function with signature 
[] (state_type &x, const input_type &u) ->  void
system_post(x,u) provides a numerical approximation of ODE solution at time tau with initial state x and input u
the result is stored in x
radius_post- lambda function with signature 
[] (state_type &r, const state_type& x, const input_type &u) -> void
radius_post(x,u) provides a numerical approximation of the growth bound for the cell (with center x, radius r) and input u
the result is stored in r

The computation proceeds in two loops. In the first loop the cornerIDs are comuted, which represents the cell indices that cover the over-approximation of the attainable set: 

corner_IDs[i*M+j][0]

= lower-left cell index of cells that cover the over-approximation of attainable set 

corner_IDs[i*M+j][1]

= upper-right cell index of cells that cover the over-approximation of attainable set

In the second loop the data members of the TransitionFunction are computed.
