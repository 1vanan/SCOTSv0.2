


Directory structure:

./src/sparse
  Contains the C++ source code in wich sparse matrices are used
  to represent the transition relation of any transition system 
  (such as an abstraction)


./mfiles
  Contains an mfile as a wrapper to the mex-file functions

./mfiles/mexfiles/
  A mex-file to read the C++ output from file 

./examples/
  Some C++ programs for testing and development

./external
  external libraries/tools
